# Scroll Security Suite

End-to-end security tooling for projects building on Scroll. Pre-deploy scanning, post-launch monitoring, and incident response. Built for the Scroll Security Subsidy Program.

## Features
- Pre-deployment contract scanning
- Real-time transaction monitoring on Scroll
- Anomaly detection (unusual transfers, access pattern changes)
- Security score for deployed contracts
- Automated incident alerts

## Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Monitoring:** Node.js event listeners, Scroll RPC
- **Analysis:** Slither, custom heuristics
- **Contracts:** Solidity (Foundry)
